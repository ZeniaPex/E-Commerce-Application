E-Commerce App
This is an E-Commerce mobile application built with Flutter. It allows users to browse and purchase products, manage their shopping cart, and complete checkout processes. The app features a modern, responsive UI and integrates with backend APIs to retrieve product data, manage user authentication, and handle orders.


Features
User Authentication: Secure login and registration with Firebase Authentication.
Product Catalog: Browse a variety of products with categories and search functionality.
Shopping Cart: Add products to the cart and proceed to checkout.
Checkout Process: Review cart items, apply discount codes, and complete payments.
Order History: View past orders and order details.
Push Notifications: Get notified about offers, promotions, and updates.
